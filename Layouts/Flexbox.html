Flexbox

Spacing

By giving the main element of this page a property of display and value of flex, all child elements (in this case divs) will move on top of each other as long as there is space to do so. This allows a flexible layout for the child elements.

These examples use a justify-content property to control the horizontal spacing and vertical spacing can be achieved by using the align-items property.

Changing the side of the window will also shring and expand (up to its size property)

Flex-basis

The flex-basis CSS property sets the initial main size of a flex item. It sets the size of the content box unless otherwise set with box-sizing
