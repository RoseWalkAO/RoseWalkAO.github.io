The Box Model

The box model is the area around an element such as this paragraph that gives it space to be properly seen. It is made up of the width & height, the content stored, the padding, a border and it's margin. Within this section, we have declared (with CSS) that:

  • The width is 50% of the screen size and the height is 60 pixels high
  • There is a space of 10px between the content and it's border. This is the padding.
  • The border is 5 pixels thick
  • The margin is 10 pixels. This creates space from other content.
  • Content - The content of the box, where text and images appear
  • Padding - Clears an area around the content. The padding is transparent
  • Border - A border that goes around the padding and content
  • Margin - Clears an area outside the border. The margin is transparent

Shorthand styling

When targeting an element's attribute, shorthand properties can be used to minimise code. For example, if we are targeting an element's border.

  • 1 value syntax - border-width: 1em - The single value represents all edges.
  • 2 value syntax - border-width: 1em 2em - The first value targets the top and bottom properties. The second targets the sides
  • 3 value syntax - border-width: 1em 2em 3em - The first value targets the top, second the sides and third the bottom.
  • 1 value syntax - border-width: 1em 2em 3em 4em - Working it's way clockwise, top, right, bottom and left are represented.
