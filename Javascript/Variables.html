Variables

Variables are used to hold bits of information and carve out a spot of memory to hold things. This information can then be retrieved later. This can be someting like information a user had put on a form, values, numbers & strings.

Basics

Variables are declared by using the 'var' keyword. By assigning a variable, it can be easier to refer to with a function.
var car = document.getElementById("car")
Instead of referring to the source, we can just refer to "car".

Arrays

Defining arrays

Arrays are variables which contain multiple values simultaneously. They can be declared in two ways:
Shorthand - var array = [50, 60, "text", 14, "more text"]
longhand - var array = []; var array = 50; var array = 60; var array = "text"; var array = 14; var array = "more text";

Individual values can be called by using zero based indexing. The first item will be 0, second 1, third 2. To call the value 60, we can do array[1]. Properties can also define information such as how long and how many variables are in an array.

Creating arrays

An array can also be created from the elements of a webpage

Organising arrays

By giving the function a method to show a list, information can be shown in different orders:

  • .sort - Alphabetical
  • .join - All joined together
  • .reverse - Reverse order
