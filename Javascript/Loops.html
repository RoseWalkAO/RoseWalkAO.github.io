Cheat sheet

Loops

While loops

While loops check a condition before excecuting. They can loop forever and cannot be escaped if the condition is always true.

Do While loops

Do while loops are only performed ONCE.

For loops

They are executed on a certain number of times depending on a variable number.

Modifying loops

Break

This special word terminates the loop as soon as it is triggered

Continue

This will skip the loops current iteration and bounce bach to the loop to start the next one.
