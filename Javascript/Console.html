If statements

If else statements

let hungerLevel = 7
if(hungerLevel >=7){
console.log('Time to eat!')
}else{
console.log('We can eat later!')
}
If hungerlevel is higher or equal to 7, it is time to eat. Otherwise we can eat later.

Logical if else statements

let mood = 'sleepy';
let tirednessLevel = 6;
if (mood==='sleepy' && tirednessLevel > 8){
console.log("It's time to sleep")
} else{
console.log("It's not bed time yet")
}
If mood EQUALS sleepy AND tiredness level is MORE THAN 8 then it is time to sleep.
|| (pipe character) can replace && to create a new If statement:
If mood EQUALS sleepy OR tiredness level is MORE THAN 8 then it is time to sleep.

Truthy and Falsy

let numberOfApples = 0;
if (numberOfApples){
console.log('Let us eat apples!'); As long as there are apples (more than 1 is true), we can eat apples!
} else {
console.log('No apples left!');
}

let tool = '';
let writingUtensil = tool || 'pen'
console.log(`The ${writingUtensil} is mightier than the sword.`);
 Tool is empty so it has no value. Since the console.log is asking for a value, writingUtensil has been given a default value of pen, filling the condition.

Shorthand coding if else with a ternary operator

hungerLevel >=7 ? console.log('Time to eat!') : console.log('We can eat later!')
mood==='sleepy' && tirednessLevel > 8 ? console.log("It's time to sleep"): console.log("It's not bed time yet")
numberOfApples ? console.log('Let us eat apples!') : console.log('No apples left!')

The above examples have been re-written to use the ternary operator instead of the if statements.
The is condition provided before the ?.
Two expressions follow the ? and are separated by a colon :.
If the condition evaluates to true, the first expression executes.
If the condition evaluates to false, the second expression executes.

Else if statements

let trafficLight = 'amber'; The value for traffic light has been set
if (trafficLight === 'red') {
console.log('Stop!'); If trafficLight is red, say "Stop!"
} else if (trafficLight === 'amber') {
console.log('Slow down.'); If trafficLight is amber, say "Slow down."
} else if (trafficLight === 'green') {
console.log('Go!'); If trafficLight is green, say "Go!"
} else {
console.log('TRAFFIC LIGHT BROKE!!!');
} If all else if statements fail the last else statement will be printed.

Switch statements

let athleteFinalPosition = 'first place';
switch (athleteFinalPosition){
case 'first place':
console.log('You get the gold medal!');
break;
case 'second place':
console.log('You get the silver medal!');
break;
case 'third place':
console.log('You get the bronze medal!');
break; } default: ''
console.log('No medal awarded.');
break; In each statement we are checking to see if it fits the athleteFinalPosition value. If it does, it's string will be printed. A default message will be printed if none of the criteria are met.
