The Document Object Model

What is the DOM?

Javascript does not exist in a vacuum. It is performed and attached to HTML web pages. The web page must be reached to get information and that page will need to respond to the code to create the interactivity.
Document is the webpage itself, Object refers to the objects that are created based on the HTML's structure and Model is the way to approach various objects.

The HTML DOM is a standard for how to get, change, add or delete HTML elements. DOM methods are actions used to perform on HTML elements. DOM properties are values we can set or change. All HTML elements are defiend as objects and are accessible through javascript. A property is a value we can retrieve or modify (adding or deleting).

In this page's HTML, the script tags are at the bottom of the page since they will take effect after the HTML elements have been loaded. Otherwise JS will attempt to target objects which don't yet exist.

Accessing the DOM

We have a few options in finding an HTML's elements: (CASE SENSITIVE)

  • ID - document.getElementsById("")
  • Tag - document.getElementsByTag Name("")
  • Class - document.getElementsByClassName("")
  • CSS selector - document.querySelectorAll("")
