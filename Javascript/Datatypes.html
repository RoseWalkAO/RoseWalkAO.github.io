Cheat sheet

Datatypes

Numbers, Strings and Booleans

Integer is a numeric value, while String is a character value represented in quotes.

aNumber = 1
aString = "1"
bString = "2"
aBoolean = true

aNumber + aNumber = 2
aNumber + aString = 11
aString + bString = 12
bString + aNumber = 21
aNumber + aBoolean = 2
aBoolean has a positive value of 1

Global methods

"parseInt" convertes the value (in this case a string) into an integer.
Now aNumber + aString = 2
"parseFloat" converts its first argument to a string

Variables

var - This is the original way to declare a variable value
let - This is a newer function and detects Variables
const - Can be grouped similar to var & let but the value always stays the same. aBoolean's true value is constant. Allowing the false value will break the calculation.

Conditional statements

if - Always works because the statement is true. If A is less than B, run the script 1.
else - If the above is false (A is more than B), run script 2.
else if - If the previous statement is false but the current one is true.
switch case - Offers a set of cases that can be triggered by a variable of a certain value.
